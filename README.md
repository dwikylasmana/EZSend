EZSend merupakan salah satu website yang berguna sebagai admin / pegawai di perusahaan jasa ekpedisi barang lokal. Website ini dibuat dengan framework Code Igniter versi 3.1.1.


Menu:
- Login
- Homepage
- Order / tampil
- Order / tambah
- Katalog
- Kontak Kami
- Riwayat Pengiriman
